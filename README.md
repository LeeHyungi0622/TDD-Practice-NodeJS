  - **(2) Mongoose :** MongoDB를 편리하게 사용하게 해주는 노드의 확장 모듈이다.

  - **(3) Jest :** 단위테스트를 위한 모듈

  - **(4) node-mocks-http :** 단위 테스트를 위한 모듈

  - **(5) supertest :** 통합 테스트를 위한 모듈
